# 🎯 לומדת אמיר"ם / אמירנט

לומדה אינטראקטיבית להכנה למבחן **אמיר"ם / אמירנט** (מבחן מיון לרמות באנגלית של מאל"ו), עם אסטרטגיית לימוד ל-4 ימים ומעקב התקדמות שנשמר בדפדפן. יעד: **פטור (134+)**.

## 🚀 הפעלה
פותחים את `index.html` בדפדפן (לחיצה כפולה) — אין צורך בהתקנה או בשרת. ההתקדמות נשמרת אוטומטית ב-`localStorage`.

## ✨ מה יש בלומדה
- 📊 **לוח בקרה** — מעקב התקדמות, ספירה לאחור למבחן וסטטיסטיקות
- 🗓️ **תוכנית 4 ימים** — משימות מסומנות עם דגש על משימות ליבה
- 📚 **מודולי תיאוריה** — מבנה המבחן, אוצר מילים, דקדוק וטכניקות פתרון לכל סוג שאלה (עם דוגמאות פתורות)
- 🗂️ **66 פלאשקארדס** לאוצר מילים עם חזרה מרווחת (Leitner)
- ✍️ **בנק תרגול — 80 שאלות** (השלמת משפטים · ניסוח מחדש · הבנת הנקרא) עם משוב מיידי והסברים
- ⏱️ **סימולציה** בתנאי זמן עם טיימר והערכת ציון
- 📓 **יומן טעויות** שמתמלא אוטומטית
- 🌙 מצב כהה/בהיר · 📱 מותאם מלא למובייל · 💾 ייצוא/ייבוא גיבוי
